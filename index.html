HTML

  • The head element contains information about the webpage
  • In the head element you can use the following metadata tags.. title, styile, base, link, meta (3.1.3)
  • The body element represents the visible content shown to the user.
  • HTML attricutes are always defined in the beginning element, never the closing element.
  • semantic HTML elements indicate the role of their content and gives structure to webpage

    CSS

  • A margin indicates how much space we want around the outside of the element
  • A padding indicates how much space we want around the content inside an element
  • Three different ways to style a webpage using CSS include inline, internal and external.
  • Most common use of CSS styling is external
  • The period preceding "card" designates this as class selector
    •

    GIT

  • Git is used for version control
  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and checkout switches to it
  • git add -A : stage all files in working branch
  • git commit -m : commit code and -m is used to associate a message with our commit
  • git push origin feature-branch-name
    •

    JAVASCRIPT

  • JavaScript has different variables able to be used (ie. String, Number, Boolean, Array, Object
  • Only strings have to be placed in quotation marks
  • JavaScript can automatically perform operations between two different types of variables
  • Strict equality uses three equal signs ===
  • ++ is a function to increase the value of a variable by one
  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
